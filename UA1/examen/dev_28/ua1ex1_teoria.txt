a) Definición de proceso. Ilustra tu solución con un ejemplo. 1 punto. 
  
Un proceso es totalmente independiente a una ejecucion de un programa. 
Cada proceso tiene una cierta memoria , lo que significa que el código de un proceso no se mezclan con los de otro codigo. 
El sistema operativo gestiona su ciclo de vida,se inicia y se ejecuata ese proceso asta que finaliza asignando recursos y componentes del sistema.
Los procesos se pueden ejecutar en paralelo o en serie, dependiendo de los recursos disponibles y de las necesidades del sistema.
Como ejemplo varios proceso se suma para poder crear la finalidad de un programa.


b) Utilizando tus propias palabras, indica la relación entre hilos y procesos. Ilustra tu solución con un ejemplo. 3 puntos.

Un solo hilo es la ejecución dentro de un proceso es decir un proceso tiene varios hilos y dentro de cada hilo se esta desarrollando una tarea. 

Los hilos se puede comunicar entre si asi hace que un proceso puede ir mas rapido. Como por ejemplo Cuando hay varios tareas en programa  como multitarea hacemos que con varios hilos vaya mas rapido el programa y se encarga cada uno de una tarea especifica.
 
Un ejemplo, el proceso tiene varios hilo y un hilo se encarga de carga la interfaz grafica y otro se encarga de recarga la pagina o de controlar el tema de descarga...

-La programacion concurrente se trata de que se que haga varias tarea a la vez es decir que se ejecucte al mismo tiempo.
Esto podemos utilizar los procesos y los hilos lo utilizamos para hace tarea simultanea es decir a la vez.
Podriamos poner un ejemplo como cuando varias persona solicitan un servicio o peticion y se ejecucte toda a la vez mediante cada hilo y proceso.
